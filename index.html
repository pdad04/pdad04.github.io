Favwod

I built FavWOD to fill a void I found while trying to keep track of workouts I was interested in doing. Until I built this, I had to rely on writing things down as a note on my phone which I didn't find useful, which ultimately lead me to not write things down. This is a simple way to enter and track workouts I want to remember.

Favwod was built with:
  • React
  • React Router
  • React Materialize
  • Firebase for data persistance and authentication

Rental Watch

I built this as a React Native iOS app. The purpose of this app is simply a way to keep track of movies that I want to watch when they become available for rent; keeping track with pen and paper or in my notes app wasn't cutting it. The app allows searching the iTunes store and adding movies to your watch list so you can see when the movie becomes available.

Rental Watch was built with:
  • React Native
  • Redux
  • IgniteCLI
